HTML Quotations

Em HTML, você pode usar elementos específicos para marcar citações dentro do seu conteúdo. Existem dois elementos principais para isso: <blockquote> para citações longas e <q> para citações curtas.
<blockquote>: Este elemento é usado para citações longas que geralmente ocupam vários parágrafos.
<q>: Este elemento é usado para citações curtas, normalmente dentro de um parágrafo.

</blockquote>
