HTML Table

As tabelas HTML são usadas para organizar dados em linhas e colunas. Elas são criadas usando as tags <table> para definir a tabela em si, <tr> para as linhas da tabela, <th> para as células de cabeçalho e <td> para as células de dados. As tabelas HTML consistem em linhas (<tr>) e células (<td> para dados, <th> para cabeçalhos) organizadas dentro da tag <table>.

<tr>
<th>Nome</th>
<th>Idade</th>
<th>País</th>
</tr>
