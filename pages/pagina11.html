HTML Colors

Em HTML, as cores podem ser especificadas usando diferentes métodos, como nomes de cores pré-definidos, códigos hexadecimais, códigos RGB ou códigos RGBA.

HTML define uma série de nomes de cores que podem ser usados diretamente em propriedades de estilo CSS. Alguns exemplos são "red" (vermelho), "blue" (azul), "green" (verde), etc.Exemplo:
<p style="color: red;">Texto vermelho</p>

Os códigos hexadecimais são representações de cores usando uma combinação de seis caracteres alfanuméricos (0-9 e A-F). Eles começam com o caractere "#" seguido de seis dígitos.Exemplo:
<p style="color: #00ff00;">Texto verde</p>

Os códigos RGB representam cores como uma combinação de valores de vermelho, verde e azul. Cada valor varia de 0 a 255.Exemplo:
<p style="color: rgb(255, 0, 0);">Texto vermelho</p>

São semelhantes aos códigos RGB, mas com um valor adicional para a transparência (alfa), variando de 0 a 1.Exemplo:
<p style="color: rgba(255, 0, 0, 0.5);">Texto vermelho semi-transparente</p>
