HTML Comments

Comentários em HTML são usados para adicionar notas ou informações dentro do código HTML que não são exibidas na página renderizada pelo navegador. Eles são úteis para documentar o código, fazer anotações para outros desenvolvedores ou temporariamente desativar parte do código sem removê-lo completamente. Comentários em HTML são escritos entre '<!-- e -->'.

<body>
<!-- Este é um comentário HTML. Ele não será exibido na página. -->
<h1>Título da Página</h1>
<p>Este é um parágrafo de texto.</p>
<!-- <p>Este parágrafo está temporariamente desativado.</p> -->
</body>
