HTML Favicon

O favicon (abreviação de "favorite icon") é um pequeno ícone associado a um site que é exibido na barra de endereço do navegador, nas guias da página e em outros locais, dependendo do navegador. Ele ajuda os usuários a identificar rapidamente um site entre várias guias abertas. O favicon é definido na seção <head> do HTML usando a tag <link> com o atributo rel="icon" e o atributo href especificando o URL do ícone. Aqui está um exemplo:
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemplo de Favicon</title>
<link rel="icon" href="favicon.ico" type="image/x-icon">
</head>
<body>
<h1>Exemplo de Favicon</h1>
<p>Este é um exemplo de como definir um favicon para o seu site.</p>
</body>
</html>
