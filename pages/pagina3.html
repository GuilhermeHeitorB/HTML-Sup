HTML Elements

Os elementos HTML são usados para estruturar e apresentar o conteúdo de uma página da web. Eles podem ser elementos de bloco, como <div>, <p>, <h1> a <h6>, <ul>, <ol>, <table>, etc., que geralmente começam em uma nova linha e ocupam todo o espaço horizontal disponível, ou elementos inline, como <span>, <a>, <strong>, <em>, que ocupam apenas o espaço necessário e não começam em uma nova linha.

Esse é um exemplo de Parágrafo:
<p>Este é um exemplo de parágrafo.</p>

Esse é um exemplo de Tabela:
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
