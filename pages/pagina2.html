HTML Basic

HTML é a espinha dorsal de uma página da web, fornecendo a estrutura básica e o conteúdo para o navegador da web renderizar. Ele consiste em uma série de elementos (ou tags) que definem diferentes partes da página, como cabeçalhos, parágrafos, imagens, links e muito mais.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document<title>
</head>
<body>
