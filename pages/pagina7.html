HTML Styles

Estilos HTML são usados para alterar a aparência de elementos HTML, como texto, imagens, ou outros elementos, diretamente no código HTML. Os estilos HTML são definidos usando o atributo style, que pode ser adicionado diretamente a um elemento HTML para controlar sua aparência. Os estilos incluem propriedades como cor, fonte, tamanho, margens, preenchimento e muito mais. Embora os estilos HTML sejam convenientes para pequenas alterações de estilo, é mais comum usar folhas de estilo em cascata (CSS) para estilos mais complexos e reutilizáveis.

Exemplo:
<h1 style="color: blue; font-family: Arial;">Título Principal</h1>
<p style="font-size: 18px; line-height: 1.5;">Este é um parágrafo de exemplo com estilos aplicados diretamente no HTML.</p>
