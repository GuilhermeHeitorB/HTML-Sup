HTML Headings

Os cabeçalhos HTML (Headings) são usados para definir os títulos ou subtítulos de uma página da web. Eles vão de <h1> a <h6>, sendo <h1> o mais importante e <h6> o menos importante. Os cabeçalhos são usados para dividir o conteúdo de uma página da web em seções e fornecer uma estrutura hierárquica. O <h1> é o título principal da página, enquanto <h2> pode ser usado para subtítulos importantes e assim por diante. Os mecanismos de pesquisa e os leitores de tela usam a hierarquia dos cabeçalhos para entender a estrutura e a importância do conteúdo.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemplo de Cabeçalhos HTML</title>
</head>
<body>
<h1>Título Principal</h1>
<h2>Subtítulo Importante</h2>
<p>Este é um parágrafo de texto.</p>
<h3>Subtítulo Menos Importante</h3>
<p>Mais um parágrafo aqui.</p>
</body>
</html>
